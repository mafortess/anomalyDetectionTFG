Dashboard Monitoring Help

Available Simulations

  • Simulation 1: High Availability Servers
    Three Ubuntu servers are monitored in separate virtual machines to simulate a high availability environment. Each server has Node Exporter installed to scrape system metrics such as CPU, RAM, storage, and network traffic. Additionally, Tshark is installed on each server to capture network traffic and send it to CloudShark for analysis. Stress-ng is used to ensure consistent CPU and RAM load across the servers, with alerts set up for deviations exceeding 20%. Notifications are sent via email, Discord, and Telegram when anomalies are detected.
  • Simulation 2: Docker Engine Monitoring
    Three Docker engines are monitored, each running three instances of docker-compose including a server, an application, and a database. Tshark captures network traffic and sends it to CloudShark. Stress-ng is used to simulate load, with alerts configured for load deviations. Metrics for each Docker engine are displayed in Grafana panels, with notifications sent via email, Discord, and Telegram for any issues.
  • Simulation 3: Energy Consumption Monitoring
    This simulation monitors energy consumption using a combination of theoretical simulation and real-time monitoring. Random energy consumption values are generated by a C script in ESP32 and scraped by Prometheus.
  • Simulation 4: Host and Container Anomaly Detection
    Hosts and containers are monitored using Isolation Forest models to detect anomalies in real-time. Data on CPU, RAM, and network traffic is collected and processed to train the models. Grafana displays metrics and anomalies, with alerts and notifications sent for detected issues.

How to Access Simulations

To access the simulations, follow these steps:

  1. Navigate to the dashboard homepage.
  2. From the main menu, select the "Simulations" tab.
  3. Choose the desired simulation from the list of available options.

Frequently Asked Questions (FAQ)

  • Q: How often is the data updated?
    A: The data is updated every 60 seconds to provide real-time insights.
  • Q: Can I export the simulation data?
    A: Yes, you can export the data in CSV format by clicking the "Export" button on the simulation page.
  • Q: What should I do if I encounter an error?
    A: If you encounter an error, please contact our support team at support@energymonitoring.com for assistance.
